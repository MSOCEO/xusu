引擎特性

Markdown 写作

本地 .md 文件即文章，front matter 管理元信息。所见即所得，无需后台编辑器。

⚡

双模架构

静态模式一键部署 GitHub Pages / Vercel；动态模式接入数据库，支持在线编辑。

🎨

双主题引擎

朱墨（书卷气）与溯光（社论风）两套主题并存，构建时选定，运行时即刻切换。

🌗

暗色模式

跟随系统或手动切换，两套主题均有独立暗色调配，夜间阅读不刺眼。
