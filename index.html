引擎特性

精于心，简于形

不堆砌功能。每一个特性都经过推敲，只保留真正提升写作与阅读体验的能力。

✍
Markdown 原生
在 content/ 下直接用 .md 文件写作。gray-matter 解析元信息，remark 渲染。文件即文章，无需数据库。
⚡
双模架构
静态模式导出纯 HTML，扔到任何服务器上直接跑。动态模式接入数据库后支持在线编辑。同一套代码按需切换。
🌙
暗色模式
跟随系统或手动切换。朱墨与溯光各自拥有独立的暗色调配方案，夜间阅读柔和不刺眼。
📤
评论系统
内置自托管评论，localStorage 存储免后端。支持地域标识、设备识别、GIF 动图。也可一键接入 Waline。
🔧
插件机制
通过 _plugins.json 在 head / bodyStart / bodyEnd 注入自定义代码。不侵入核心，扩展无限。
🚀
<5s 构建速度
Next.js 静态导出，构建产物是一堆 HTML + JS + CSS 文件。零运行时依赖，极致轻量。
