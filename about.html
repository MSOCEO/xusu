叙溯是什么

叙溯——一边叙说，一边追溯。

一个为书写而生的开源博客引擎。完整的前端渲染管道 + 双主题系统，让你专注于内容本身。

核心特性

  • 双模架构：静态模式一键部署 GitHub Pages / Vercel，动态模式接入数据库在线编辑
  • 双主题引擎：朱墨（书卷气）与溯光（社论风）内置于产物，构建时选定，运行时即刻切换
  • Markdown 原生：content/ 目录写 .md 文件，gray-matter + remark 管线渲染
  • 暗色模式：跟随系统或手动切换，两套主题独立色调配，夜间不刺眼
  • 零运行时依赖：纯 Next.js 静态导出，无数据库也可运行

技术栈

Next.js 15 · TypeScript · Tailwind CSS 4 · remark
